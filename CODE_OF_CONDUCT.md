# 社区行为准则

Image2 AI 生图系统希望保持友好、务实、尊重事实的开源协作氛围。参与 issue、PR、讨论和技术支持时，请遵守以下原则。

## 我们鼓励

- 使用清晰、尊重、具体的表达方式。
- 围绕可复现问题、代码事实和用户体验讨论。
- 接受不同部署环境、技术背景和使用目标带来的差异。
- 在反馈问题时保护自己和他人的隐私、密钥和凭证。

## 我们不接受

- 人身攻击、骚扰、歧视、威胁或持续挑衅。
- 在公开渠道发布他人的隐私、密钥、Cookie、Token 或生产凭证。
- 明知会造成破坏仍公开传播可直接利用的安全漏洞细节。
- 与当前项目无关的大量广告、刷屏或恶意引流。

维护者可以编辑、隐藏或删除违反准则的内容，并在必要时限制相关账号继续参与项目协作。
