# 支持说明

如果你在部署、升级或二次开发 Image2 AI 生图系统时遇到问题，可以优先通过 GitHub Issue 反馈。可复现的问题越完整，越容易被定位和修复。

## 推荐反馈方式

- Bug、部署失败、构建失败、移动端布局异常：请创建 GitHub Issue，并选择对应模板。
- 使用咨询、紧急部署协助或非公开日志：可通过 README 中的技术支持二维码联系维护者。
- 安全漏洞、密钥泄露风险或支付回调安全问题：请按 `SECURITY.md` 私下反馈。

## 提交问题时请提供

- 项目版本或 commit。
- 部署方式：源码部署、Docker Compose、已有 PostgreSQL 或其他方式。
- 操作系统、Node.js、npm、PostgreSQL、Docker 版本。
- 复现步骤、预期结果和实际结果。
- 关键日志、截图或录屏。

## 请勿公开提交

- API Key、Cookie、Token、数据库密码、支付密钥。
- 真实用户邮箱、订单号、支付凭证、后台登录信息。
- 远程仓库、服务器或对象存储的鉴权信息。
