# 安全策略

Image2 AI 生图系统涉及用户登录、后台配置、图片上传、Provider API Key、积分账户和支付配置。请不要在公开 issue、PR、截图或日志中暴露可利用的安全细节。

## 支持版本

当前优先维护 `main` 分支和最新 GitHub Release。旧版本如存在安全问题，建议先升级到最新版本再复测。

## 报告安全问题

如果你发现以下问题，请私下联系维护者：

- API Key、Cookie、Token、数据库密码或支付密钥可能泄露。
- 后台权限绕过、会话伪造、未授权访问。
- 上传文件绕过、路径穿越、任意文件读取或写入。
- 支付回调伪造、积分异常入账或订单状态篡改。
- 其他可能影响自部署用户数据安全的问题。

推荐方式：

- 使用 README 中的技术支持二维码私下反馈。
- 如果 GitHub Security Advisory 已启用，也可以通过 GitHub 私有漏洞报告提交。

请在报告中提供影响范围、复现步骤、相关版本和必要的最小日志。请不要附带真实用户隐私、完整密钥或可直接滥用的生产凭证。

维护者会尽力确认、修复并在必要时更新 `CHANGELOG.md` 和 `UPGRADE.md`。
