# 更新日志

本项目遵循面向开源自部署用户的发布记录。维护者发布新版本时，请把用户需要知道的功能变化、修复、数据库迁移和环境变量变化写在这里。

## 未发布

- 修复 Docker 构建时 TDesign 邮箱输入类型和友情链接解析的 TypeScript 报错。
- 修正 Docker Compose 图片 volume 挂载路径，使默认本地存储持久化到 `/app/public/storage/generated` 和 `/app/public/storage/uploads`。
- 升级脚本新增旧版图片目录备份和回填，避免从旧 Compose 路径升级时丢失容器内 `/app/public/storage` 图片。
- 修复创作页移动端生成失败后错误提示、选项控件和底部导航可能错位或溢出的问题。
- 修复创作页桌面端失败态右侧结果预览栏因 sticky 定位导致与左侧表单错位的问题。

## v0.1.3 - 2026-06-18

- 后台新增中转站推荐一级菜单，并保留明心 AI、极狗中转站两个 Logo 跳转入口。
- 后台移动端改为左侧抽屉导航和右侧用户入口，优化统计卡、用户表、任务表和作品表在手机端的排版。
- 修复后台 TDesign 组件水合错误，避免下拉箭头在服务端和客户端渲染 class 不一致导致控制台报错。
- 修复后台移动端表格操作列固定、积分输入控件错位、用户表内容被遮挡等问题。
- 优化创作页移动端生成后的布局，避免 DeepSeek 整理结果和底部导航发生遮挡。
- 调整创作页桌面端布局，将任务信息和积分说明移动到创作表单下方，手机端展示顺序保持不变。
- 优化前台首页背景、Logo/Favicon 展示、能力卡片和页脚在桌面及手机端的视觉表现。

## v0.1.2 - 2026-06-18

- 优化 TDesign 企业官网首页首屏，增加产品工作台预览和作品结果宫格。
- 新增首页作品结果预览带，公开作品不足时自动使用内置提示词示例兜底。
- 将产品能力区调整为“一大三小”结构，强化从中文描述到视觉资产沉淀的主张。
- 优化适用场景区，改为“适合谁 / 输入 / 产出”的企业官网表达。

## v0.1.1 - 2026-06-18

- 新增前台模板切换能力，支持 `TDesign 企业官网风` 和旧版 `玻璃拟 App 风` 在后台切换。
- 将 TDesign 前台首页调整为企业官网首页，用户功能页继续保留应用工作台结构。
- 丰富企业官网首页内容，增加产品可信能力、适用团队、生产流程、运营能力、场景矩阵和底部转化区。
- 拆分普通用户登录、普通用户注册和管理员登录入口，新增 `/signup` 与 `/admin/signin`。
- 前台充值支持展示所有已启用且配置完整的支付渠道，并使用后台自定义显示名称。
- 普通用户新注册需要邮箱验证码，已有用户登录和管理员登录不受影响。
- 注册验证码增加 60 秒发送冷却，避免同一邮箱频繁发送。
- 支付宝当面付兼容裸应用私钥/支付宝公钥格式，避免创建订单时暴露 OpenSSL 内部错误。
- 支付宝当面付待支付订单增加主动查单兜底，异步回调未到达时也可在刷新状态后自动入账，并记录支付诊断事件。
- 优化支付宝当面付主动查单体验，新订单前 2 分钟更快确认，后台诊断区分自动查单与手动查单。
- 支付宝当面付下单后增加服务端短轮询确认，用户离开用户中心页面后仍会在短时间内自动查单入账。
- 优化用户中心充值订单轮询，订单未变化时不再反复替换二维码区域，降低二维码闪烁。
- 用户中心待支付订单自动刷新时额外拉取单笔订单状态，并在页面重新可见时立即刷新，提升到账状态同步速度。
- 用户中心账单状态接口和前端轮询增加防缓存处理，避免 CDN 或浏览器复用旧的待支付状态。
- 修复用户中心自动刷新后刚创建的充值订单可能从前台列表消失的问题。
- 新增 `UPGRADE.md`，说明源码部署、Docker Compose 部署和二开用户的升级流程。
- 新增 `scripts/update.sh`，提供源码部署、完整 Docker Compose、仅 Web Docker Compose 三种常用升级入口。
- 更新 README，补充升级说明、版本发布建议，并修正 Docker Compose 使用 `.env.production` 的命令写法。

## v0.1.0 - 2026-06-17

- 开源第一版，包含中文 AI 生图 WebUI、提示词库、DeepSeek 提示词润色、生图任务、作品历史、积分账户、后台配置和 Docker Compose 部署基础。
- 默认生图 Provider 为 `openai`，通过统一后端接口调用模型能力。
- `chatgpt_web` Provider 仅作为实验性预留，未配置时返回明确的 disabled 错误。
