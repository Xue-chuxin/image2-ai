# 更新日志

本项目遵循面向开源自部署用户的发布记录。维护者发布新版本时，请把用户需要知道的功能变化、修复、数据库迁移和环境变量变化写在这里。

## 未发布

- 第三方账号绑定/解绑：用户中心「账号安全」页新增「第三方账号绑定」区块，已登录用户可把 GitHub / Google 账号绑定到当前账号或解绑（复用现有 OAuth 回调地址，无需新增回调登记）。绑定时校验第三方账号未被他人占用、同一渠道不重复绑定；解绑时若账号未设置登录密码且这是最后一种登录方式则拒绝，避免锁死账号。管理员账号不支持第三方绑定。

- 会员权益落地：有效会员可享三项权益，均在控制台「安全与存储 → 会员权益」配置——① 出图积分折扣（按 % off 减免，最低仍收 1 积分）；② 每日赠送积分（每天首次出图时自动发放一次，幂等不重复）；③ 出图频率上限提升（普通用户固定每 10 分钟 10 次，会员按配置值）。折扣在下单估价时生效，冻结/结算/退款全链路一致；充值页展示「会员专享权益」清单。`Subscription` 表新增 `lastDailyGrantAt` 字段（需执行数据库迁移）。

- 会员卡（月卡/年卡）：积分套餐新增「套餐类型」，可设为「会员卡（周期）」并配置有效天数（月卡填 30、年卡填 365）。用户购买会员卡后即时发放套餐积分，同时按天数续期会员有效期（未到期时叠加顺延）；充值页展示当前会员状态与到期提醒（剩余 7 天内或已到期高亮提示），到期后手动购买续费。管理沿用现有在线支付渠道，无自动代扣。新增数据表 `Subscription`，`CreditPackage`/`RechargeOrder` 新增 `packageType`/`durationDays` 字段（需执行数据库迁移）。

- 第三方登录（OAuth）：登录/注册页新增「使用 GitHub / Google 登录」入口（控制台「安全与存储 → 第三方登录」配置，默认关闭）。已验证邮箱会自动关联到同邮箱的既有账号，否则创建新用户并发放新用户积分；管理员账号不支持第三方登录。回调地址为 `{站点地址}/api/auth/oauth/{github|google}/callback`，站点地址取 `NEXT_PUBLIC_SITE_URL`。新增数据表 `OAuthAccount`（需执行数据库迁移）。相关环境变量：`OAUTH_GITHUB_CLIENT_ID`、`OAUTH_GITHUB_CLIENT_SECRET`、`OAUTH_GOOGLE_CLIENT_ID`、`OAUTH_GOOGLE_CLIENT_SECRET`。

- 创作页：生成任务状态改为 SSE 实时推送（`GET /api/generation/jobs/{id}/events`），到账更快、请求更省；浏览器不支持或连接异常时自动回退到原有 2s 轮询。离开页面或重新提交会自动断开连接。

- 内容安全：违禁词拦截之外新增「语义审核」（控制台「安全与存储 → 内容安全」开关，默认关闭）。开启后关键词放行的提示词会再调用 DeepSeek 兼容通道做语义判断（复用「生图通道」的 DeepSeek 配置，可单独指定审核模型）。模型不可用/超时时自动放行，不阻断正常生成。相关环境变量：`MODERATION_TIMEOUT_MS`（可选，默认 15000）。

- 存储：新增对象存储支持（S3 兼容协议，一套配置覆盖 AWS S3 / 阿里云 OSS / 腾讯云 COS）。控制台「安全与存储」页签新增 Endpoint / Bucket / Region / AccessKey / 路径风格寻址配置；生成图与参考图上传后落对象存储并回写可访问 URL，支付凭证等私有文件仍走鉴权回源路由（不公开直链）。切回 local 本地存储行为不变。相关环境变量：`STORAGE_PROVIDER`、`STORAGE_ENDPOINT`、`STORAGE_BUCKET`、`STORAGE_REGION`、`STORAGE_FORCE_PATH_STYLE`、`STORAGE_ACCESS_KEY_ID`、`STORAGE_SECRET_ACCESS_KEY`。

- 正式开放参考图生图：控制台「系统设置 → 生图通道」新增「开放参考图生图」开关（默认关闭，也可用环境变量 `REFERENCE_IMAGES_ENABLED` 开启）。开放后创作页支持上传最多 4 张参考图，「再次生成」会完整回填参考图。
- 参考图视觉分析改走所配置的 OpenAI 兼容通道（原先硬编码官方域名，中转通道 Key 会 401 导致分析静默失效），分析模型新增「参考图视觉分析模型」配置项（默认 gpt-4o，留空关闭分析）。
- Stability AI 图生图的引擎段改用后台配置的模型名（原先硬编码 stable-diffusion-xl-1024-v1-0）。
- ChatGPT 网页版通道不支持参考图，带参考图提交时会明确报错提示切换通道，不再静默忽略。
- 安全加固：参考图上传改为服务端读取文件真实内容（magic bytes）判定类型，不再信任客户端声明的 Content-Type，伪造类型的非图片文件会被拒绝。
- 参考图新增过期清理机制：超过保留天数且未被任何生成任务引用的参考图会被软删除并删除磁盘文件；上传成功后惰性触发（进程内节流）。新增「参考图保留天数」配置项（控制台「生图通道」，默认 30 天，也可用环境变量 `REFERENCE_IMAGE_RETENTION_DAYS` 设置）。

- 前台新增暗色模式：顶栏主题按钮支持「浅色 / 暗色 / 跟随系统」三态切换，本地记忆并防止刷新闪烁，桌面与移动端均可用。
- 生成任务轮询在离开创作页或重新提交时自动停止，不再在后台持续请求。
- 控制台系统设置页首次加载显示骨架屏，取代空白等待。

- 性能：后台配置读取增加内存缓存（30 秒），大幅减少每次页面请求对配置表的重复查询；控制台设置页与全站首屏加载更快。修改配置后即时生效。
- 性能：提示词润色不再为读取一个配置项而执行完整的后台诊断（含数据库探测）。
- 性能：后台用户列表的统计计数改为批量聚合查询，用户较多时打开更快（原先每个用户单独查 4 次）。
- 性能：生成图与参考图的回源支持长效缓存与 304 协商缓存，重复浏览走浏览器/磁盘缓存，减轻服务器图片流量。

- 安全：修复"未设置密码的账号可被任意密码接管"的问题——此类账号改为拒绝密码登录，并收紧管理员初始化逻辑，不再静默提权已注册的普通用户或覆盖其密码。
- 安全：API 错误响应不再把数据库/内部异常原文透给前端，仅透传业务提示，其余返回通用文案并记服务端日志。
- 安全：支付凭证文件（storage 的 payments 目录）改为需要本人或管理员会话才能访问，未授权返回 404；系统自检新增"凭证是否暴露在 public 目录"的检查项。
- 支付：修复订单超时过期后收到成功支付却"钱付了积分不到账"的问题——过期订单迟到支付现在可正常入账；已取消订单收到支付会拦截并记诊断供人工处理。
- 支付：取消订单改为原子操作，消除与支付回调竞态时把已支付订单误标为已取消的可能；取消已支付订单会明确提示无法取消。
- 积分：修复冻结积分在重复结算/退款时可能被减为负数或双倍返还的问题（条件更新 + 任务失败状态守卫）。

## v0.2.0 - 2026-07-06

本版本为大规模 UI 重构：前台换新为浅色侧边栏应用布局，用户中心与管理后台迁移到基于 Vben Admin 的控制台。升级前请阅读 UPGRADE.md 对应章节。

- 管理后台整体迁移到控制台：运营概览（实时统计）、系统设置（六页签全量配置）、用户管理、任务管理、图片管理（生成作品+运营精选）、上传管理、账单管理（支付渠道/套餐/订单）、支付诊断、系统自检、中转站推荐，均基于现有 /api/admin/* 接口。
- 下线 Next.js 旧版 /admin 页面与 TDesign 依赖（/admin 自动跳转控制台，包体积显著减小）；管理员登录并入控制台统一登录页。
- 新增 GET /api/admin/dashboard 与 GET /api/admin/relays 接口。
- 控制台上线用户中心：积分总览、在线充值（渠道选择/扫码支付/自动到账轮询）、充值订单管理（继续支付/手动查单/取消）、积分流水、账号安全（改密），前台 /account 及顶栏充值入口改为跳转控制台。
- 新增 /api/console/billing/* 与 /api/console/user/password 接口；新增按用户查询积分流水的能力；控制台计费接口同时接受管理员与普通用户会话。
- 生产环境控制台静态产物由 Next.js 托管在 /console 路径（新增 npm run build:console 构建命令）。
- 新增基于 Vben Admin 5 (Vue 3 + Ant Design Vue) 的控制台应用（console/ 目录），用于承载用户中心与新版管理后台；登录与前台账号互通（HTTP-only Cookie 会话，前台已登录时免登录进入）。
- 新增 /api/console/* 适配接口（登录/登出/用户信息/权限码），普通用户与管理员按角色显示控制台菜单。
- 前台整体重构为浅色侧边栏应用布局（左侧导航 + 顶部搜索/积分/账户），首页改为"画廊广场"（推广 banner + 应用广场 + 作品瀑布流），视觉风格全面更新。
- 新增应用中心、智能助手、更多工具三个占位页面，展示后续规划能力，导航先行对齐完整形态。
- 下线旧版 glass_app / tdesign_workspace 双模板体系，后台"前台模板"配置不再影响前台展示（后台自身样式不受影响）。
- 顶栏新增全局作品搜索（跳转首页作品区检索）、积分余额与充值快捷入口。
- 修复"再次生成"丢失画幅比例、质量、张数、英文提示词和负面词参数的问题，现在会完整回填到创作页。
- 修复账户下拉菜单无法点击外部关闭的问题。
- 旧灵感页 /prompts 并入首页画廊广场（自动跳转），移除无入口的 /prompts/[slug] 示例详情页。
- 清理前台死代码与未使用依赖（react-bits 动效组件、motion/gsap/ogl），globals.css 由 6388 行精简到约 440 行，后台样式隔离到 admin-legacy.css。
